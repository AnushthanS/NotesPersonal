17. The following function is expected to compute the factorial of a positive number n. Is the given code error free? If not, then using one sentence state the reason why it will not work and also give the corrected code.
long fact(int n) {
long k=1;
do
k*=fact(i--);
while(n>0);
return k; }


Answer: No i variable declared, a loop using recursion gives segmentation fault. Also, i-- will give an infinite loop. Plus the value of k will reset with every function call.

fixed :

long fact(int n){
    long k=n;
    if(n==0 || n==1) return 1;
    k*=fact(n-1);
    return k;
}
